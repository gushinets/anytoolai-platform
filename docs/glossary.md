# Glossary

- **Action Definition**: Product-neutral typed logical operation.
- **Action Configuration**: Product/scenario-specific configuration for an action type.
- **Artifact**: Stored workflow/action result or intermediate output.
- **CE**: Chrome Extension.
- **Frontend**: Delivery surface such as CE or web mirror.
- **Kernel Demo**: Internal MVP-A smoke product used only to prove the platform runtime.
- **Handoff**: Backend-owned user-confirmed transfer from one scenario/product to another.
- **Paywall Intent**: Runtime record that quota exhaustion led to paywall or waitlist interest.
- **Product Bundle**: Product-specific config package connected to the platform through platform-sdk.
- **Provider Gateway**: Platform-owned boundary for all provider/model calls.
- **Scenario Chain**: Linked scenario sessions connected by handoff or continuation.
- **Scenario Session**: Stateful journey record for a user in a product scenario.
- **Workflow**: Backend-defined chain of action configurations.
