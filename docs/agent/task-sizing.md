# Task Sizing

Prefer tasks that fit one PR and one execution plan.

## Good agent task

- Add one config validator.
- Add one action definition.
- Add one API endpoint plus tests.
- Add one CE component using ce-kit.

## Too large

- Build all workflow runtime.
- Implement Freelancer Suite.
- Make product quality good.
- Add billing.

Split large tasks into contracts, storage, service, API, tests, docs.
